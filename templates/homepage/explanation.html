Image placeholders generated automatically from the URL. Edit the location and save the resulting image, or use curl. The URL contains the following parts, of which only the size is not optional.

  • the hex value for the background colour (or transparent)
  • the hex value for the line colour (or none)
  • the hex value for the text colour (or none)
  • text to use instead of the default
  • the size of the image
